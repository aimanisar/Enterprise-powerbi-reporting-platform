# Enterprise Power BI Reporting Platform

## Overview

Developed an enterprise Power BI reporting solution to support financial performance monitoring, exposure analysis, and management reporting.

The solution consisted of 15 interactive reports built using Power BI, SQL Server, DAX, and Power Query. Users could dynamically explore data using slicers, filters, and drill-through functionality while access was controlled through role-based permissions.

Due to confidentiality requirements, company-specific data and report assets have been anonymised.

---

## Project Highlights

* Developed 15 Power BI reports and dashboards
* Built semantic models using 4–5 fact tables and 20+ dimension tables
* Created DAX measures and time-intelligence calculations
* Implemented Row-Level Security (RLS)
* Implemented Role-Based Access Control (RBAC)
* Contributed to Power BI embedding using ASP.NET Core
* Developed interactive reporting with drill-through, filters, and slicers
